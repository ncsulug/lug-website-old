The LUG Wiki uses WikiCreole 1.1 markup, with a few modifications. Here's the most important stuff:

Inline Markup

//italics// emphasis
**strong** strong
`inline code` inline code

Headings

== Heading 2
=== Heading 3
==== Heading 4

Links

Links to URLs and wiki pages take the form [[target|text]]. If the text to display is the same as the target, you can just use [[target]].

Link to users' profiles with [[~username]].

Code

Enclose code blocks in {‌{‌{ and }‌}‌}. To syntax-highlight it, put #!language at the front. For example: 

{‌{‌{
#!python
print "Hello, world!"
}‌}‌}
